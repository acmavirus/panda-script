```go
POST /api/deploy/:name/auto-deploy // Enable webhook

// Stats
GET  /api/projects/stats       // Get all project counts
```

### Project Directories
- Node.js projects: `/opt/nodejs-apps/`
- Python projects: `/opt/python-apps/`
- Java projects: `/opt/java-apps/`
- Deployment configs: `/etc/panda/deployments/`

- Node.js: PM2 (cluster mode)
- Python: Systemd (python-{name}.service)
- Java: Systemd (java-{name}.service)
